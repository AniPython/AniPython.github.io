获取文字坐标 findWords()#

AndroidBot.findWords(words, options)#

查找文字

参数:

  • words (string) – 要查找的文字

返回:

AndroidBotMain.find_text(text: str, region: Optional[Tuple[float, float, float, float]] = None, algorithm: Optional[Tuple[int, int, int]] = None, scale: float = 1.0) → List[Point]#

查找文字所在的坐标，返回坐标列表（坐标是文本区域中心位置）

参数:

  • text – 要查找的文字；

  • region – 识别区域，默认全屏；

  • algorithm – 处理图片/屏幕所用算法和参数，默认保存原图；

  • scale – 图片缩放率，默认为 1.0，1.0 以下为缩小，1.0 以上为放大；

返回:

坐标列表（坐标是文本区域中心位置）

参见

find_image(): region 和 algorithm 的参数说明

获取屏幕文字 getWords()#

AndroidBot.getWords(options)#

获取屏幕文字

返回:

Promise.<string> – 失败返回null，成功返回手机屏幕上的文字

AndroidBotMain.get_text(region: Optional[Tuple[float, float, float, float]] = None, algorithm: Optional[Tuple[int, int, int]] = None, scale: float = 1.0) → List[str]#

通过 OCR 识别屏幕中的文字，返回文字列表

参数:

  • region – 识别区域，默认全屏；

  • algorithm – 处理图片/屏幕所用算法和参数，默认保存原图；

  • scale – 图片缩放率，默认为 1.0，1.0 以下为缩小，1.0 以上为放大；

返回:

文字列表

参见

find_image(): region 和 algorithm 的参数说明
