创建TextView控件 createTextView()#

AndroidBot.createTextView(id, text, x, y, width, height)#

创建TextView控件

参数:

  • id (number) – 控件ID，不可与其他控件重复

  • text (string) – 控件文本

  • x (number) – 控件在屏幕上x坐标

  • y (number) – 控件在屏幕上y坐标

  • width (number) – 控件宽度

  • height (number) – 控件高度

返回:

Promise.<boolean> – 成功返回true，失败返回 false

创建WebView控件 createWebView()#

AndroidBot.createWebView(id, url, x, y, width, height)#

创建WebView控件

参数:

  • id (number) – 控件ID，不可与其他控件重复

  • url (string) – 加载的链接

  • x (number) – 控件在屏幕上x坐标，值为-1时自动填充宽高

  • y (number) – 控件在屏幕上y坐标，值为-1时自动填充宽高

  • width (number) – 控件宽度，值为-1时自动填充宽高

  • height (number) – 控件高度，值为-1时自动填充宽高

返回:

Promise.<boolean> – 成功返回true，失败返回 false

AndroidBotMain.create_web_view(_id: int, url: str, x: int = -1, y: int = -1, width: int = -1, height: int = -1) → bool#

创建WebView控件
