使用chatgpt chatgpt()#

WindowsBot.chatgpt(model, prompt, maxTokens, temperature, stop)#

chatGpt

参数:

  • model, (string) – 指定使用的模型,”text-davinci-003”text-curie-001text-babbage-001text-ada-001和自定义微调模型

  • prompt, (string) – 问题

  • maxTokens, (number) – 最大令牌数，大约 3个字符1个令牌，1个汉字2个令牌

  • temperature, (number) – 浮点型，温度，调节结果的创意程度，0一般为单一结果， 1创意度更高

  • stop, (string) – 可选参数，停止结果输出标志，一般用在微调模型上，例如 stop = ‘[“END”]’

返回:

Promise.<string> – 失败返回null, 成功返回json对象 text 返回的答案内容 finish 为true回答结束，false 还有未输出的答案。我们可以继续 prompt + 输出的答案 获取后续内容，直到finish为true

使用chatgpt编辑模式 chatgptEdit()#

WindowsBot.chatgptEdit(model, input, instruction, maxTokens, temperature)#

chatGpt编辑模式

参数:

  • model, (string) – 指定使用的模型， “text-davinci-edit-001””code-davinci-edit-001”

  • input, (string) – 输入要编辑的内容

  • instruction, (string) – 提示如何去编辑

  • maxTokens, (number) – 最大令牌数，大约 3个字符1个令牌，1个汉字2个令牌

  • temperature, (number) – 浮点型，温度，调节结果的创意程度，0一般为单一结果， 1创意度更高

返回:

Promise.<boolean> – 失败返回null, 成功返回json对象 text 返回的答案内容 finish 为true回答结束，false 还有未输出的答案。我们可以继续 prompt + 输出的答案 获取后续内容，直到finish为true

// 代码示例
// 初始化
let bRet = await windowsBot.initNLP("xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx");
if (!bRet)
    // return;

// chatgpt
let result = await windowsBot.chatgpt("text-davinci-003", "怎么称呼你？", 256, 0.7);
console.log(result["text"]);

// chatgpt 循环获取答案
let prompt = "请介绍下Aibote的应用场景，不少于500字";
let isFinish;
do {
    let result = await windowsBot.chatgpt("text-davinci-003", prompt, 256, 0.7);
    isFinish = result["finish"];
    if (!isFinish)
        prompt += result["text"];

    console.log(result["text"]);
} while (!isFinish)

// chatgptEdit
let result = await windowsBot.chatgptEdit("text-davinci-edit-001", "你好码？", "修正错误文字", 256, 0);
console.log(result["text"]);
