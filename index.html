欢迎#

Aibote 是江西爱伯特科技自主研发的一款纯代码RPA办公自动化框架，支持 Android、Browser 和 Windows 三大平台。 框架免费、API和接口协议开源，个人、企业商用零费用。 以socket tcp接口协议通信方式命令驱动，支持任何一门计算机语言调用。 OEM或特殊接口定制等商业合作联系：QQ：1600381943

Aibote python 是由 AiBot.py 模块作者: 河畔(810975099) 制作而成，提供给 python 开发者调用。 python版本是基于aibote底层api开源协议制作而成，封装格式以 node.js 为向导，涵盖aibote自动化平台所有功能。
